# Chinese Bits - 需求文档

## 项目目标
帮助外国人快速学习日常中文，掌握1000句常用中文表达。

## 核心功能

### 1. 1000句常用中文
- 覆盖日常生活多种场景
- 中英文对照显示
- 支持发音功能

### 2. 场景分类
- 问候与介绍
- 购物
- 餐饮
- 交通
- 住宿
- 旅游
- 就医
- 工作与学习
- 社交与娱乐

### 3. 学习功能
- 逐句发音
- 收藏常用句子
- 学习进度追踪
- 复习模式

## 技术规划
- 前端：React / Vue
- 后端：Node.js / Python
- 发音：使用 TTS API
- 数据存储：JSON / SQLite

## 数据结构
```json
{
  "id": 1,
  "chinese": "你好",
  "pinyin": "nǐ hǎo",
  "english": "Hello",
  "scene": "问候",
  "audio": "audio/001.mp3"
}
```
